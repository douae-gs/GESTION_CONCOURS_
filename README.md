# Gestion de Concours ENSAO

Ce projet est une application web simple en PHP et MySQL permettant de gérer des concours et des candidatures d'étudiants.

## Description du projet

L'application permet d'automatiser et de simplifier le processus d'inscription aux concours. Elle propose deux espaces distincts selon le rôle de l'utilisateur : un espace pour les candidats afin de postuler, et un espace pour les administrateurs afin de gérer l'ensemble des offres de concours.

## Fonctionnalités

* Authentification : Inscription et connexion sécurisées pour les utilisateurs.
* Espace Candidat : Consultation des concours disponibles et soumission de candidatures.
* Espace Admin : Ajout de nouveaux concours, modification de leur statut (ouvert, fermé, en cours) et suppression.
* Profil : Consultation des informations personnelles et possibilité de modifier son nom.
